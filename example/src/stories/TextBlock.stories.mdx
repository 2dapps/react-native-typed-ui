## Passing Text Props

Since the props of `TextBlock` are text styles, if you want to pass any props to the underlying `<Text/>` component, you can pass it via the `text` prop.

<Canvas>
  <Story name="Passing Text Props">
    <TextBlock text={{ onPress: () => alert('Clicked Text') }}>
      Click on this text!
    </TextBlock>
  </Story>
</Canvas>
