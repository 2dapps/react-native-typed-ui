# Theme object

A Theme object represents the configuration of your design system. The object should be strictly typed & has the following 11 types of styling configuration for your app.

- [colors](#colors)
- fonts
- fontSizes
- fontWeights
- lineHeights
- letterSpacings
- spacing
- zIndices
- iconSizes
- borderRadius
- shadows

The theme object should extend the following interface

```tsx
import type { TextStyle, ViewStyle } from 'react-native';

export default interface theme {
  colors: {
    [color: string]: ViewStyle['backgroundColor'];
  };
  fonts: {
    [font: string]: TextStyle['fontFamily'];
  };
  fontSizes: {
    [fontSize: string]: TextStyle['fontSize'];
  };
  fontWeights: {
    [fontWeight: string]: TextStyle['fontWeight'];
  };
  lineHeights: {
    [height: string]: TextStyle['lineHeight'];
  };
  letterSpacings: {
    [spacing: string]: TextStyle['letterSpacing'];
  };
  spacing: {
    [spacing: string]: number;
  };
  zIndices: {
    [index: string]: number;
  };
  iconSizes: {
    [size: string]: number;
  };
  borderRadius: {
    [radius: string]: ViewStyle['borderRadius'];
  };
  shadows: {
    [index: string]: 2 | 4 | 6 | 8 | 12 | 16 | 24;
  };
}
```

Since Typed UI requires the default theme to be strongly typed, this requires the use of typescript. If you are using javascript, you will have to add a type declaration (`.d.ts`) file for your theme object.

## colors

You can specify your color pallete in this configuration. After you specify your color pallete, it will be accessible in all the style props that uses color such as `color`, `backgroundColor`, `borderColor` etc.

The default theme has the following color configuration

export const boxConfig = {
  height: 100,
  width: 100,
  alignItems: 'center',
  justifyContent: 'center',
