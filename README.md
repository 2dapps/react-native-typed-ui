# react-native-typed-ui

A strongly typed themeable design system for react native

## Installation

```sh
npm install react-native-typed-ui
```

## Usage

```js
import TypedUi from "react-native-typed-ui";

// ...

const result = await TypedUi.multiply(3, 7);
```
